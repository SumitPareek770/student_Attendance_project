🎓 Student Attendance Management System
A simple and responsive Student Attendance Management System built using HTML, CSS, and JavaScript. This project helps teachers and administrators efficiently manage student attendance records through an easy-to-use web interface.

🚀 Features

📋 Add student records

✅ Mark students as Present or Absent

📊 Real-time attendance tracking

🔍 Search and filter student records

💾 Data persistence using Local Storage

📱 Responsive design for mobile and desktop devices

🎨 Clean and user-friendly interface

⚡ Fast and lightweight application

🛠️ Technologies Used

HTML5 – Structure of the application
CSS3 – Styling and responsive design
JavaScript (ES6) – Application logic and functionality
Local Storage – Store attendance data in the browser
